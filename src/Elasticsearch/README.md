Elasticsearch Component
==============

The Elasticsearch component is the elasticsearch adapter for the shopwell/core.
It contains the indexing of entities and an adapter for the entity search.

This repository is considered **read-only**. Please send pull requests
to our [main Shopwell repository](https://github.com/shopwell/shopwell).
